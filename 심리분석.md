1. 성격 (Personality)

기반: Big Five Personality Traits

핵심 trait (연속값으로 저장)
Openness (개방성)
상상력 / 예술적 관심 / 지적 호기심
Conscientiousness (성실성)
계획성 / 자기통제 / 책임감
Extraversion (외향성)
사교성 / 활동성 / 자극 추구
Agreeableness (친화성)
공감 / 협력성 / 신뢰성
Neuroticism (정서 안정성)
불안 / 스트레스 민감도 / 감정 기복
추가 확장 (선택)
Honesty-Humility (정직성) → HEXACO Model
Risk-taking tendency (위험 선호)
Impulsivity (충동성)
2. 가치관 (Values)

기반: Schwartz Value Theory

핵심 가치 축
Self-Direction (자율성)
Achievement (성취)
Power (권력/영향력)
Security (안정)
Conformity (규범 준수)
Tradition (전통)
Benevolence (이타성)
Universalism (보편적 가치)
Hedonism (쾌락)
Stimulation (자극/새로움)
실무용 파생 카테고리
돈 vs 의미 중심
개인주의 vs 집단주의
단기 보상 vs 장기 가치
효율 vs 공정성
3. 행동 (Behavior)

기반: Dual Process Theory + 실제 UX 데이터

(1) 의사결정 스타일
분석형 vs 직관형
빠름 vs 신중함
정보 탐색 깊이 (얕음 / 깊음)
옵션 비교 성향
(2) 리스크 & 선택 패턴
Risk tolerance (위험 회피/선호)
Loss aversion (손실 회피)
실험 성향 (새로운 것 시도 여부)
(3) 행동 습관
계획형 vs 즉흥형
지속성 (꾸준함)
멀티태스킹 vs 집중형
(4) 인터랙션 패턴 (챗봇에서 매우 중요)
질문 스타일 (짧음 / 상세함)
반복 질문 여부
피드백 반응 (수용 / 반박)
대화 길이 선호
(5) 학습 스타일
텍스트 vs 예시 vs 단계별 설명
빠른 요약 vs 깊은 설명 선호
4. 취향 (Preferences / Interests)

👉 실제 서비스에서는 가장 강력한 신호

(1) 관심사 클러스터
기술 / IT
투자 / 금융
건강 / 운동
여행 / 음식
콘텐츠 (영화, 게임 등)
(2) 소비 취향
프리미엄 vs 가성비
브랜드 선호
신제품 vs 검증된 제품
(3) 콘텐츠 스타일
짧은 콘텐츠 vs 긴 콘텐츠
시각적 vs 텍스트
실용 vs 재미
(4) 미적 취향
미니멀 vs 화려함
전통 vs 현대
감성 vs 기능 중심

좌절 민감도
동기 상태

6. 컨텍스트 (상황 기반 보정)

👉 이거 없으면 분석 다 틀어짐
